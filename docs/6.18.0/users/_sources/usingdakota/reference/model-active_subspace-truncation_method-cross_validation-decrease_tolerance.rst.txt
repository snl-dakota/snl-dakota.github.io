decrease_tolerance
""""""""""""""""""


Tolerance for cross-validation error stabilization


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* REAL

- *Default:* 1.0e-6


**Description**


Select smallest subspace where change in cross-validation error falls
below specified tolerance.
