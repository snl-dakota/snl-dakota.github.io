global_model_pointer
""""""""""""""""""""


Pointer to model used by global method



**Topics**


block_pointer


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* STRING


**Description**


``global_model_pointer`` can be used to specify a model for use with the Dakota
method named by the ``global_method_name`` specification. The argument is a
string that refers to the :dakkw:`model-id_model` label of the desired model.
