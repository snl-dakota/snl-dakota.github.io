""""""""""""""
percent_change
""""""""""""""


Define the convergence criterion for JEGA methods


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* REAL

- *Default:* 0.1


**Description**


The ``percent_change`` is the threshold beneath which convergence is
attained whereby it is compared to the metric value computed.
