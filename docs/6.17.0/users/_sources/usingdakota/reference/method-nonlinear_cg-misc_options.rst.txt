.. _method-nonlinear_cg-misc_options:

""""""""""""
misc_options
""""""""""""


Options for nonlinear CG optimizer


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* STRINGLIST

- *Default:* no misc options


**Description**


List of miscellaneous string options to pass to the experimental
nonlinear CG solver (see NonlinearCGOptimizer.cpp in the Dakota source
code for available controls).  Includes controls for step sizes,
linesearch control, convergence, etc.
