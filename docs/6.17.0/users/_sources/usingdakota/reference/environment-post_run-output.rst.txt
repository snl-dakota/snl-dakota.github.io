output

Base filename for post-run mode data output



**Topics**


dakota_IO


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* STRING

- *Default:* no post-run specific output written


**Description**


(For future expansion; not currently used by any
methods.) Specify a base filename to which Dakota will write any
post-run output data. This option is overridden by any command-line

-post_run arguments.
