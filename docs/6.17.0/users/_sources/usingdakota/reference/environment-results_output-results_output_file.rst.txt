results_output_file
"""""""""""""""""""


The base file name of the results file



**Topics**


dakota_output


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* STRING

- *Default:* dakota_results


**Description**


The default filename is ``dakota_results``.txt for text output  and
``dakota_results``.h5 for ``h5HDF5``. The ``dakota_results`` stem can be
changed using this keyword. Do not include the extension; these will
be added by Dakota.
