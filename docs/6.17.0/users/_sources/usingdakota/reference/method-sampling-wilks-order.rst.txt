.. _method-sampling-wilks-order:

"""""
order
"""""


The order of the statistics to use when determining sample sizes
for random sampling using Wilks order statistics.


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* INTEGER


**Description**


*Default Behavior*

The default order is 1.



**Examples**



.. code-block::

    method
      sampling
        sample_type random
        wilks
          order 2
