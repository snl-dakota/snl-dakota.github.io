.. _model-active_subspace-truncation_method-cross_validation-exhaustive:

""""""""""
exhaustive
""""""""""


Assess all admissible subspace dimensions


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None

- *Default:* on


**Description**


Assess cross-validation error for all admissible subspace dimensions,
even if a particular dimension meets specified cross-validation
tolerance(s).
