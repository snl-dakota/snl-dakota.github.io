""""""
active
""""""


use statistics for the active expansion within multifidelity refinement


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


When performing adaptive refinement in multifidelity UQ
methods, this option adapts based on the statistics for the active
model level/fidelity.  This approach avoids some computational overhead in some
cases by avoiding expansion roll-up.
