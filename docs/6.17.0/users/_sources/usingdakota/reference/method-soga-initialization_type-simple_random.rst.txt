.. _method-soga-initialization_type-simple_random:

"""""""""""""
simple_random
"""""""""""""


Create random initial solutions


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


``simple_random`` creates initial solutions with random variable
values according to a uniform random number distribution. It gives no
consideration to any previously generated designs.
