""""""""""""""""""""""""
local_search_probability
""""""""""""""""""""""""


Probability of executing local searches


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* REAL


**Description**


The ``local_search_probability`` setting is an optional specification for
supplying the probability (between 0.0 and 1.0) of employing local search
to improve estimates within the global search. Its default value is 0.1.
