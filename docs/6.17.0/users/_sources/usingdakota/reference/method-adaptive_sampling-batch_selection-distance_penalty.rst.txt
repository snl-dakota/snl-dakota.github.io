.. _method-adaptive_sampling-batch_selection-distance_penalty:

""""""""""""""""
distance_penalty
""""""""""""""""


Add a penalty to spread out the points in the batch


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


In this strategy, the highest scoring candidate is selected and then all
remaining candidates are re-scored with a distance penalization factor
added in to the score.
