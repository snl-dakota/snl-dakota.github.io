""""""""""""""
mutation_range
""""""""""""""


Set uniform offset control for discrete parameters


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* INTEGER

- *Default:* 1


**Description**


The ``mutation_range`` is used to control ``offset_uniform`` mutation used for
discrete parameters. The replacement discrete value is the original value
plus or minus an integer value up to ``mutation_range``.
