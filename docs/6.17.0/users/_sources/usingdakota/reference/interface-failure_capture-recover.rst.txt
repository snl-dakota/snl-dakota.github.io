.. _interface-failure_capture-recover:

"""""""
recover
"""""""


Substitute dummy values for the responses


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* REALLIST


**Description**


When a simulation failure is detected, substitute the provided dummy
function values in the response.  Gradient and Hessian are not
supported.
