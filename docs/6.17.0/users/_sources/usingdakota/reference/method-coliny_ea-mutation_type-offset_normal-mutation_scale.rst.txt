""""""""""""""
mutation_scale
""""""""""""""


Scales mutation across range of parameter


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* REAL

- *Default:* 0.1


**Description**


The ``mutation_scale`` specifies a scale factor which scales continuous mutation
offsets; this is a fraction of the total range of each dimension, so
``mutation_scale`` is a relative value between 0 and 1.
