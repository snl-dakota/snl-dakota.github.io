.. _method-coliny_ea-initialization_type-flat_file:

"""""""""
flat_file
"""""""""


Read initial solutions from file


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* STRING


**Description**


``flat_file`` allows the initial population to be read from a flat
file. If ``flat_file`` is specified, a file name must be given.
