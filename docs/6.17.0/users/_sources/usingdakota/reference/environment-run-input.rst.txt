Base filename for run mode data input



**Topics**


dakota_IO


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* STRING

- *Default:* no run specific input read


**Description**


(For future expansion; not currently used by any
methods.) Specify a base filename from which Dakota will read any run
input data, such as parameter sets to evaluate. This option is
overridden by any command-line -run arguments.
