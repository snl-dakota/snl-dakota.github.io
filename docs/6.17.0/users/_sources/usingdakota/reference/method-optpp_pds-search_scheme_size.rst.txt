search_scheme_size

Number of points to be used in the direct search template


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* INTEGER

- *Default:* 32


**Description**


The ``search_scheme_size`` is defined for the PDS method to specify the
number of points to be used in the direct search template.
