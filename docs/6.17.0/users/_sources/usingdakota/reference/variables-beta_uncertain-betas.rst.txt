Second parameter of the beta distribution


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* buv_betas 

- *Arguments:* REALLIST


**Description**


Specifies the list of :math:`\beta`  parameters to define the distributions of the beta random variables. Length must match the other parameters and the number of beta random variables.
