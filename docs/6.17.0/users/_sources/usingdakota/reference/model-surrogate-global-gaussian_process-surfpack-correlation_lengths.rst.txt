correlation_lengths
"""""""""""""""""""


Specify the correlation lengths for the Gaussian process


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* REALLIST

- *Default:* internally computed correlation_lengths


**Description**


Directly specify ``correlation_lengths`` as a list of N real
numbers where N is the number of input dimensions.
