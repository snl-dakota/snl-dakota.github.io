Second parameter of the Frechet distribution


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* fuv_betas 

- *Arguments:* REALLIST


**Description**


Specifies the list of :math:`\beta`  parameters to define the distributions of the Frechet random variables. Length must match the other parameters and the number of Frechet random variables.
