"""""""""""""""
full_covariance
"""""""""""""""


Display the full covariance matrix


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


With a large number of responses, the covariance matrix can
be very large. ``full_covariance`` is used to
force Dakota to output the full covariance matrix.
