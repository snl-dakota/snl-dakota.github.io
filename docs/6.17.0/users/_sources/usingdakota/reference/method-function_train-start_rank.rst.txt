""""""""""
start_rank
""""""""""


The initial rank used for the starting point during a rank adaptation.


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* rank 

- *Arguments:* INTEGER

- *Default:* 2


**Description**


The ``adapt_rank`` procedure increments the rank on each iteration, starting
from the rank defined by ``start_rank``

*Default Behavior*

Default is 2.
