"""""""
aprepro
"""""""


Write parameters files in APREPRO syntax



**Topics**


file_formats


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* dprepro 

- *Arguments:* None

- *Default:* standard parameters file format


**Description**


The
format of data in the parameters files can be modified for direct
usage with the APREPRO pre-processing tool
:cite:p:`Sja92`
using the ``aprepro`` specification

Without this keyword, the parameters file are written in
DPrePro format. :ref:`DPrePro <interfaces:dprepro-and-pyprepro>` is
a utility included with Dakota.
