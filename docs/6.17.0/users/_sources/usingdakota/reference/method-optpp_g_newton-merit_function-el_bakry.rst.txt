el_bakry
""""""""


El-Bakry merit function


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


The "el_bakry" merit function is the L2-norm of the first order
optimality conditions for the nonlinear programming problem. The cost
per linesearch iteration is n+1 function evaluations. For more
information, see :cite:p:`ElBak1996`.
