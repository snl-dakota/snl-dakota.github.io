First parameter of the gumbel distribution


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* guuv_alphas 

- *Arguments:* REALLIST


**Description**


Specifies the list of :math:`\beta`  parameters to define the distributions of the gumbel random variables. Length must match the other parameters and the number of gumbel random variables.
