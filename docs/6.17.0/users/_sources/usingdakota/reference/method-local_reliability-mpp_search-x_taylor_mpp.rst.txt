""""""""""""
x_taylor_mpp
""""""""""""


X-space Taylor series approximation with iterative updates



**Topics**


reliability_methods


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


This mpp_search option starts with an x-space Taylor series at the uncertain variable means,
but iteratively updates the Taylor series approximation at each MPP prediction until the MPP converges.
This option is commonly known as the AMV+ method.
