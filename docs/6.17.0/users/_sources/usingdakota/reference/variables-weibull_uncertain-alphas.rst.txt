alphas
""""""


First parameter of the Weibull distribution


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* wuv_alphas 

- *Arguments:* REALLIST


**Description**


Specifies the list of :math:`\alpha`  parameters to define the
distributions of the Weibull random variables.

Length must match the other parameters and the number of Weibull random variables.
