""""""""
max_rank
""""""""


Maximum subspace dimension to consider


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* INTEGER

- *Default:* number fullspace vars


**Description**


In cross-validation, only consider subspace dimensions from 1 to
``max_rank``.
