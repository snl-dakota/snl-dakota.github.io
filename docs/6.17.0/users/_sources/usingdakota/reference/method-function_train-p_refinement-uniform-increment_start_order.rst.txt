"""""""""""""""""""""
increment_start_order
"""""""""""""""""""""


candidate generation by advancement of starting basis order


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


Within adaptive refinement processes, this specification selects the
advancement of ``start_order`` to create new refinement candidates.
It is typically used in the absence of ``adapt_order`` to more rigidly
define a refinement sequence.
