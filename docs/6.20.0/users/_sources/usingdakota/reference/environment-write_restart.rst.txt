"""""""""""""
write_restart
"""""""""""""


Base filename for restart file write



**Topics**


dakota_IO, command_line_options


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* STRING

- *Default:* dakota.rst


**Description**


Specify a base filename for the restart file Dakota
should write.  This option is overridden by any command-line

-write_restart option.
