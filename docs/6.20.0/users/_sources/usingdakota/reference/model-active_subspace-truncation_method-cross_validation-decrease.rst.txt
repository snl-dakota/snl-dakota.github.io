""""""""
decrease
""""""""


Choose subspace where cross-validation error stabilizes


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


Select smallest subspace where change in cross-validation error to
next larger subspace falls below specified tolerance.
