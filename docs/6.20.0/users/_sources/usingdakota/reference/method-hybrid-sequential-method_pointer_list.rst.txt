method_pointer_list
"""""""""""""""""""


Pointers to methods to execute sequantially or collaboratively



**Topics**


block_pointer


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* STRINGLIST


**Description**


``method_pointer_list`` specifies by name the methods that are to be executed by
a ``hybrid`` ``sequential`` or ``hybrid`` ``collaborative`` method. Its argument is
a list of strings that refer to method blocks by name (i.e. to their
:dakkw:`method-id_method` labels).
