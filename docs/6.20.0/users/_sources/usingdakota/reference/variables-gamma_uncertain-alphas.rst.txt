alphas
""""""


First parameter of the gamma distribution


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* gauv_alphas 

- *Arguments:* REALLIST


**Description**


Specifies the list of :math:`\alpha`  parameters to define the distributions of the gamma random variables. Length must match the other parameters and the number of gamma random variables.
