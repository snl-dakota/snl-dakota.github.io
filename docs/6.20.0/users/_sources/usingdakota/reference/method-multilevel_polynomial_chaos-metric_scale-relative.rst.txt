relative

relative scaling of statistical metrics when adapting UQ surrogates


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


When adapting a UQ surrogate model, the ``relative``
option indicates that the change is measured relative to the current
reference value (which evolves during the refinement), such that the
convergence tolerance controls the relative change.
