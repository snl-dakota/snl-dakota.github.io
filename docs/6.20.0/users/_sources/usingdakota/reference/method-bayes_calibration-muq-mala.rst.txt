""""
mala
""""


Metropolis-adjusted Langevin algorithm




**Topics**


bayesian_calibration



.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None

- *Default:* dram


**Description**


When the user selects 'bayesian_calibration muq' in the Dakota input file,
the occurrence of the 'mala' keyword will cause Dakota to use the MALA
algorithm (:cite:p:`RobTwe96`) from the MUQ library.
