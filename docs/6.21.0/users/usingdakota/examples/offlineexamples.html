Offline Examples

Outside of Sandia, the Examples Library currently is available only in Dakota source and binary packages. In source packages, it is located in the dakota-examples folder. In binary packages, it is locatedin share/dakota/examples/official. We plan to mirror the repository externally in the very near future; to be notified, email dakota-announce@software.sandia.gov and ask to be subscribed.

See also:

  • The Dakota GUI is a convenient and powerful way to view and find examples. It includes a text-based search interface to quickly search for examples.
