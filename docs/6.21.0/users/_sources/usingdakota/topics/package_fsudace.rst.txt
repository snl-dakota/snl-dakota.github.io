Package: FSUDace
================

The Florida State University Design and Analysis of Computer
Experiments (FSUDace) library provides the following DACE techniques:
quasi-Monte Carlo sampling ( ``fsu_quasi_mc``) based on the Halton
sequence ( ``halton``) or the Hammersley sequence ( ``hammersley``),
and Centroidal Voronoi Tessellation ( ``fsu_cvt``).
