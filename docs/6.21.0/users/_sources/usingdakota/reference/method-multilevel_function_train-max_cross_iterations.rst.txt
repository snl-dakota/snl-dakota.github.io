max_cross_iterations
""""""""""""""""""""


Maximum number of iterations for cross-approximation during a rank adaptation.


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* INTEGER

**Description**
