""""""""""""""""""""
original_constraints
""""""""""""""""""""


Use the constraints directly


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


The surrogate constraints in the approximate subproblem can be
selected to be surrogates of the original constraints (
``original_constraints``) or linearized approximations to the surrogate
constraints ( ``linearized_constraints``), or constraints can be omitted
from the subproblem ( ``no_constraints``).
