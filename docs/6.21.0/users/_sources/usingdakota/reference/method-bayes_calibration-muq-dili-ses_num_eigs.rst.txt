ses_num_eigs
""""""""""""


Parameter for the stochastic eigensolver used by the DILI MCMC algorithm in MUQ



.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* INTEGER

- *Default:* 2


**Description**


The maximum number of eigenvalues to compute. Used as stopping criteria. The default value is '2'.
