"""""""""""
mfmc_groups
"""""""""""


Throttle multilevel BLUE to only search over MFMC groups



.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


The number of groups in ML BLUE is reduced to include only the
ordered hierarchical group definitions from MFMC.  The number of
groups is equal to the total number of models.
