.. _model-active_subspace-normalization-local_gradient:

""""""""""""""
local_gradient
""""""""""""""


Normalize each gradient sample by its norm


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


For each reponse function :math:`f_i` , normalize each gradient sample
:math:`\nabla^j f_i`  by its two-norm :math:`\|\nabla^j f_i\|`
