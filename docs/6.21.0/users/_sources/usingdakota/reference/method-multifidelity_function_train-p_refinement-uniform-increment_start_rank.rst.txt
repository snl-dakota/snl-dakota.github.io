""""""""""""""""""""
increment_start_rank
""""""""""""""""""""


candidate generation by advancement of starting rank


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


Within adaptive refinement processes, this specification selects the
advancement of ``start_rank`` to create new refinement candidates.
It is typically used in the absence of ``adapt_rank`` to more rigidly
define a refinement sequence.
