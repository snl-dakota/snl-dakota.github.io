"""""""""""""""""""""
convergence_tolerance
"""""""""""""""""""""


Expected improvement convergence tolerance


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* REAL

- *Default:* 1.e-12


**Description**


The Efficient Global Optimization method will terminate if the
expected improvement is below the specified tolerance for two
subsequent iterations.
