""""""
output
""""""


Base filename for run mode data output



**Topics**


dakota_IO


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* STRING

- *Default:* no run specific output written


**Description**


(For future expansion; not currently used by any
methods.) Specify a base filename to which Dakota will write any run
output data (typically parameter, response pairs). This option is
overridden by any command-line -run arguments.
