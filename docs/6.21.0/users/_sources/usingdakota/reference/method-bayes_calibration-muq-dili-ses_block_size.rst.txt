""""""""""""""
ses_block_size
""""""""""""""


Parameter for the stochastic eigensolver used by the DILI MCMC algorithm in MUQ



.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* INTEGER

- *Default:* 2


**Description**


The number of new random matvecs to compute when the current eigenvalues do not satisfy the stopping criteria. The default value is '2'.
