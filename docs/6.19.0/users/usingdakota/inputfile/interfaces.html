Interfaces

The interface specification in a Dakota input file controls all the details of function evaluations. The mechanisms currently in place for function evaluations involve interfacing with one or more computational simulation codes, computing algebraic mappings, or a combination of the two.

It is highly recommended that, at minimum, you are familiar with the general strategy by which Dakota interfaces with external simulation models before delving into this section.
