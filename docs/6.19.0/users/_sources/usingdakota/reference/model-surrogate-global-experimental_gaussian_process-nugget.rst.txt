""""""
nugget
""""""


Value for the fixed nugget parameter


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* REAL

- *Default:* None


**Description**


This keyword is used to specify a value for the nugget parameter. Its presence
will turn off nugget estimation.

*Default Behavior*
No nugget when this keyword is absent. Alternatively a nugget may be estimated as
part of the regression procedure.
