find_nugget
"""""""""""


Use regression to estimate the nugget.


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* INTEGER

- *Default:* None


**Description**


Estimate a nugget term in the Gaussian process Gram matrix.

*Default Behavior*
Off by default. The user may alternatively specify a fixed nugget.
