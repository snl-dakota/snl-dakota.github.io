"""""""""""""""
solution_target
"""""""""""""""


Stopping criteria based on objective function value


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* solution_accuracy 

- *Arguments:* REAL

- *Default:* -DBL_MAX


**Description**


``solution_target`` is a termination criterion. The algorithm will terminate when the function value falls below solution_target.
