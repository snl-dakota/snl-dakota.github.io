.. _variables-histogram_bin_uncertain-counts:

""""""
counts
""""""


Frequency or relative probability of each bin


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* huv_bin_counts 

- *Arguments:* REALLIST


**Description**


The ``counts`` list of real values gives the frequency or relative
probability for each bin in a ``histogram_bin_uncertain``
specification.  These are paired with the specified
:dakkw:`variables-histogram_bin_uncertain-abscissas`. See
:dakkw:`variables-histogram_bin_uncertain` for details.
