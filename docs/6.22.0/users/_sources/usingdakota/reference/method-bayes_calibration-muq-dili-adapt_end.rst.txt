"""""""""
adapt_end
"""""""""


Parameter for the DILI MCMC algorithm in MUQ



.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* INTEGER

- *Default:* -1


**Description**


No LIS updates will occur after this number of MCMC steps. If negative, the LIS will continue to be updated until the end of the chain. The default value is '-1'.
