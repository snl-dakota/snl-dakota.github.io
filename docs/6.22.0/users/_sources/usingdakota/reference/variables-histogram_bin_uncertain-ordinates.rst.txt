"""""""""
ordinates
"""""""""


Ordinates specifying a "skyline" probability density function


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* huv_bin_ordinates 

- *Arguments:* REALLIST


**Description**


The ``ordinates`` list of real values defines the profile of a
"skyline" probability density function by pairing with the specified
:dakkw:`variables-histogram_bin_uncertain-abscissas`.  See
:dakkw:`variables-histogram_bin_uncertain` for details.
