"""""""""""""
lis_tolerance
"""""""""""""


Parameter for the DILI MCMC algorithm in MUQ



.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* REAL

- *Default:* 0.1


**Description**


Lower bound on eigenvalues used to construct local and global likelihood informed subspaces. Must be larger than 'hess_tolerance'. The default values is '0.1'.
