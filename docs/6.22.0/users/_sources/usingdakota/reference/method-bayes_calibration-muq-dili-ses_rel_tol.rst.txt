ses_rel_tol
"""""""""""


Parameter for the stochastic eigensolver used by the DILI MCMC algorithm in MUQ



.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* REAL

- *Default:* 0.001


**Description**


Fraction of the largest eigenvalue used as stopping criteria. The default value is '0.001'.
