"""""""""""""
x_multi_point
"""""""""""""


MPP search for local reliability based on QMEA multi-point approximation in u-space


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


Local reliability methods that search for the most probable point
of failure (MPP) may use a surrogate-based optimization procedure.
The ``x_multi_point`` surrogate option creates a QMEA approximation
in the original random variable space ("x-space") from accumulating
data from multiple expansion points.

This capability is \b experimental.
