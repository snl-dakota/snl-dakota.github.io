""""""""""""
hessian_type
""""""""""""


Parameter for the DILI MCMC algorithm in MUQ



.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* STRING

- *Default:* GaussNewton


**Description**


The type of posterior Hessian to use. Either "Exact" or "GaussNewton". The default value is "GaussNewton".
