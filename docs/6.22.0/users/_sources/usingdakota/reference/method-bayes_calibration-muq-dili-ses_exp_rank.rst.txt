""""""""""""
ses_exp_rank
""""""""""""


Parameter for the stochastic eigensolver used by the DILI MCMC algorithm in MUQ



.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* INTEGER

- *Default:* 2


**Description**


The expected number of eigenvalues that are larger than the the tolerances. This is used to decide on the initial number of samples to compute. The default value is '2'.
