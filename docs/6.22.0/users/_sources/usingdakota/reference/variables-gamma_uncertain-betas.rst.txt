Second parameter of the gamma distribution


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* gauv_betas 

- *Arguments:* REALLIST


**Description**


Specifies the list of :math:`\beta`  parameters to define the distributions
of the gamma random variables.
Length must match the other parameters and the number of gamma random variables.
