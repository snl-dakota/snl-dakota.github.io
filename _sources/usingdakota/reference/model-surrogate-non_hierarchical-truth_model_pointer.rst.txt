truth_model_pointer
"""""""""""""""""""


Pointer to specify a "truth" model, from which to construct a surrogate



**Topics**


block_pointer


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* actual_model_pointer 

- *Arguments:* STRING


**Description**


This must point to a model block, identified by :ref:`model-id_model<model-id_model>`.
That model will be run to generate training data, from which a
surrogate model will be constructed.

See :ref:`topic-block_pointer<topic-block_pointer>` for details about pointers.
