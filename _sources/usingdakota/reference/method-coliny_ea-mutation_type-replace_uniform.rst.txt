"""""""""""""""
replace_uniform
"""""""""""""""


Replace coordinate with randomly generated value


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


The ``replace_uniform`` mutation type generates a replacement value for a
coordinate using a uniformly distributed value over the total range for that
coordinate.
