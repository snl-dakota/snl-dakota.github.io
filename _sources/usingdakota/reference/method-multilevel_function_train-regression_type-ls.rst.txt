""
ls
""


Use least squares solver for forming function train approximations by regression


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


Least squares is the standard solver for forming function train approximations
based on regression.  This option does not include regularization.
