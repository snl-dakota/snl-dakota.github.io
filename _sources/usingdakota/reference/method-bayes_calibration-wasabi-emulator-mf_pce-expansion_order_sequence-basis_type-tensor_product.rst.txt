""""""""""""""
tensor_product
""""""""""""""


Use a tensor-product index set to construct a polynomial chaos expansion.


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


Use a tensor-product index set to construct a polynomial chaos expansion. That is for a given order :math:`p`  keep all terms with :math:`d` -dimensional multi index :math:`\mathbf{i}=(i_1,\ldots,i_d)`  that satisfies

.. math:: 

   \max \,(i_1,\ldots,i_d) \le p
