""""
grid
""""


Deprecated grid computing interface


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


Interface Dakota directly to a grid (distributed) computing engine.
This deprecated capability was historically used for interfaces with
IDEA and JAVASpaces in the past and was intended as a placeholder for
future work with Condor and/or Globus services. It is not currently
operational.
