.. _method-richardson_extrap-estimate_order:

""""""""""""""
estimate_order
""""""""""""""


Compute the best estimate of the convergence order from three points


.. toctree::
   :hidden:
   :maxdepth: 1



**Specification**

- *Alias:* None

- *Arguments:* None


**Description**


The ``estimate_order`` option is the simplest option.
For each of
the refinement factors, it evaluates three points along the
refinement path and uses these results to perform an estimate of
the convergence order for each response function.
